# NETFLIX-DATA
# Netflix Data Analysis Dashboard (Power BI)

This project is a Power BI dashboard built using the Netflix dataset to analyze movies and TV shows available on Netflix.

# Project Overview
The dashboard provides insights into:
- Total number of titles on Netflix
- Distribution of Movies vs TV Shows
- Top 10 countries producing Netflix content
- Year-wise content trend
- Content ratings and type-based filtering

# Tools Used
- Power BI
- Microsoft Excel / CSV (Dataset)

# Key Features
- Interactive slicers for Year, Rating, and Type
- Netflix-themed dashboard design
- KPI cards for quick insights
- Clean and user-friendly visuals
